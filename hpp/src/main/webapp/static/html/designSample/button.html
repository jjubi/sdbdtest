검색 폼 상단 버튼

  • 상단 버튼은 항상 그룹형태 div class="btn-group" 으로 감싸주세요.
  • 상단에 있는 버튼들은 btn-sm 클래스를 추가해 주시면 작은 버튼으로 변경됩니다.

상세페이지 하단 버튼

  • 목록보기는 항상 왼쪽으로 고정
  • div class="card-wrap-footer" 으로 감싸주세요.

테이블 안 버튼

  • 테이블 안에 버튼은 항상 작게!!!
  • 버튼들은 btn-sm 클래스를 추가해 주시면 작은 버튼으로 변경됩니다.
